Description

Dashboard at port 8088 doesn't open in the host browser from a Codespace or VS Code devcontainer. devcontainer.json uses appPort which publishes at Docker level but does not trigger VS Code port tunneling. Need to add forwardPorts: [8088] alongside appPort so VS Code Dev Containers extension auto-forwards port 8088 to the host. Recurring issue.
