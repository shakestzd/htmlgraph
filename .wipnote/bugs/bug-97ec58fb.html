Yolo guard false positive: blocking infrastructure/setup commands as 'external research required'

Todo Medium Priority

Caused By:

  • feat-6c3e6d60

Part Of:

  • trk-be293476

Description

Bash commands like mkdir -p ~/.gotest or mkdir -p /workspaces/go-tests are blocked by checkYoloBashResearchGuard if no research has been done. These are often necessary setup for tests (especially in noexec environments) and shouldn't be blocked as 'modifying files outside the project' in the same way as destructive edits.
