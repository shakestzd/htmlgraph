wipnote sh — compressing bash wrapper for agents

Todo Medium Priority

Description

wipnote sh '' runs the command via bash -c, captures stdout+stderr, applies compression (strip ANSI, dedup consecutive identical lines, drop progress bars, cap lines with truncation trailer), and exits with the child exit code. Flags: --max-lines, --no-dedup, --raw. Agent prompt updates to recommend it are deferred to a follow-up commit.
