Dashboard Observability Architectural Overhaul

Planned High Priority Created: 2026-03-07

Parallel execution: Stream A (JSONL attribution) and Stream B (live feed) run simultaneously in separate git worktrees

Overview

Three-stream parallel overhaul. Stream A replaces hook heuristics with JSONL parentToolUseID attribution. Stream B fixes the SSE 404 and replaces full-table-swap with delta/prepend. Streams A and B are fully independent (different files) and run in separate git worktrees simultaneously. Stream C (tests) merges both. Stream D (long-term) runs independently anytime.

Context

Hook-based attribution uses dead code (Method 2), SubagentStart doesn't fire for background agents, and parallel agents race to claim task_delegation rows. The JSONL transcripts Claude Code already writes have perfect attribution via parentToolUseID chains. The SSE endpoint returns 404. Full tbody replacement destroys expanded rows. All four problems have independent solutions in separate files — they can be fixed in parallel.
