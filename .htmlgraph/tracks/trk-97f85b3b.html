TrackSpecPlan

Passive Session Ingestion (AgentsView-inspired)

Planned High Priority Created: 2026-03-05

Overview

Add passive ingestion of native AI tool session files (Claude Code, Gemini CLI, Copilot CLI, Cursor, OpenCode) to complement active hook-based tracking. Inspired by AgentsView capabilities.

Requirements

⏳ Auto-discover session directories for supported AI tools

must-have

⏳ Parse native session file formats (JSONL, JSON) into HtmlGraph events

must-have

⏳ Store ingested sessions in SQLite alongside hook-recorded sessions

must-have

⏳ Render ingested sessions as HTML graph nodes

must-have

⏳ Support incremental ingestion (only new/changed files)

must-have

⏳ CLI command: htmlgraph ingest [--tool claude|gemini|copilot|cursor|opencode] [--watch]

must-have

⏳ FTS5 full-text search across all ingested session content

must-have

⏳ Session DAG/fork detection for branching relationships

must-have

⏳ Filesystem watcher for live updates without hooks

must-have
