Description

Feature badges in the activity feed should use color to indicate category (dashboard=blue, fix=red, plan=amber, feat=green) instead of showing the prefix in the title text. Strip prefixes like 'Dashboard:', 'Fix:', 'Plan:' from the displayed title and use the prefix to select badge color. The prefix metadata is preserved in the feature title — just not shown in the badge.
