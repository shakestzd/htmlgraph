Description

Rearchitect the plan system so plans are created first from a topic/title (not from existing tracks). The CRISPI skill populates design content, open questions, and proposed slices. On finalize, the approved plan generates the track with features, dependency edges, and question answers baked into feature descriptions. The existing track-to-plan path becomes secondary for retroactive planning.
