Track completion with remaining features — allow 'shipped' status with linked future work

Todo Medium Priority

Part Of:

  • trk-97f85b3b

Description

Tracks need a state between in-progress and done: 'shipped' means core work is complete but some features remain as future work. When completing a track with todo features, prompt the user: move remaining features to a new track, or mark the track as shipped (not done). The shipped status shows in the dashboard with a count of remaining items. Prevents the current pattern of either leaving tracks open forever or closing them and orphaning features.
