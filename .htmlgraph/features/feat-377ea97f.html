Description

Create `internal/registry/` with a single file `registry.go`. Define `Entry` struct: `{ProjectDir, Name, GitRemoteURL, LastSeen, ID string}` where `ID` is the first 8 characters of the SHA256 hash of `ProjectDir`, computed once on first upsert and stored in the entry. Define `Registry` struct wrapping the file path and a `[]Entry` slice. Implement `Load(path string) (*Registry, error)` — reads and JSON-decodes the file; returns empty registry (not error) if file does not exist. Implement `(*Registry) Save() error` — JSON-encodes with indentation, writes atomically via a tempfile + `os.Rename()` pattern (write to `.tmp`, then rename) to the registry path, ensuring crash-safe writes. Implement `(*Registry) Upsert(dir, name, remoteURL string)` — finds existing entry by `ProjectDir` (filepath.Clean comparison), updates `LastSeen` / `GitRemoteURL` / `Name` in-place or appends a new entry with a freshly computed `ID`; uses `time.Now().UTC().Format(time.RFC3339)` for `LastSeen`. Implement `(*Registry) List() []Entry` returning a copy of entries. Implement `(*Registry) Prune() []string` — removes entries whose `filepath.Join(entry.ProjectDir, ".htmlgraph")` does not exist via `os.Stat`; returns list of pruned dirs. Expose `DefaultPath() string` helper returning `~/.local/share/htmlgraph/projects.json` (resolved via `os.UserHomeDir`). Add `OpenReadOnly(dbPath string) (*sql.DB, error)` — opens SQLite with `file:?mode=ro&_busy_timeout=5000` URI; does NOT call `ApplyPragmas` or `CreateAllTables`; returns a read-only `*sql.DB` for use by cross-project queries. No external dependencies — stdlib only (database/sql + modernc.org/sqlite already in go.mod).
