Description

Single-command bundle that emits a markdown briefing for an agent picking up a work item. Output: claim command, branch-sync state, work-item description + steps, code-surface helpers (derived from track-aligned blame), file paths (with package qualifiers), recent same-track commits for stylistic continuity, and any open questions on related plan slices. Once shipped, agent prompts collapse from manual context bundling (track IDs, helper signatures, file globs, sync state) to: 'run htmlgraph context-pack , then implement.' Eliminates ~half of the per-agent orientation overhead. Depends on existing internal/blame, internal/migrate, and the work-item store. Out of scope: any LLM-judged synthesis — pack output is purely derived data + raw markdown, not summarized.
