Description

Replace the current broken dagre graph layout with an Obsidian-style force-directed graph using D3 force simulation. Nodes as circles colored by type (track=green, feature=blue, bug=red, spike=amber), sized by connectivity. Edges as lines. Clusters form naturally by connectivity. Zoom/pan/click to navigate. Hover shows title+status. Use the new graph query APIs (hubs, orphans, reachable) to drive the visualization. This is a key demo for the LinkedIn launch.
