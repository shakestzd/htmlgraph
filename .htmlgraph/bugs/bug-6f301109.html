YOLO mode fails silently because htmlgraph find is broken — agent loses all track context

Done Medium Priority

Caused By:

  • bug-6b9d363a

Part Of:

  • trk-3719d8f3

Description

Root cause: htmlgraph find parses ID prefixes (trk-, feat-, bug-) as collection names and fails with 'unknown collection'. Since yolo mode uses find as the entry point for context discovery, the agent concludes the track does not exist and has no work items to execute. Workaround: use track show / feature show directly. Fix: feat-7fa61af2 in trk-c26a29ab.
