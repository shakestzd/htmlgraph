    Key Implementation Details¶

    System Prompt Persistence Flow¶

    1. SessionStart Hook Triggered
    2. Claude Code triggers SessionStart hook

    3. Hook receives session info (source, session_id)

    4. Prompt Loading

    5. Hook script loads .claude/system-prompt.md

    6. Falls back to HtmlGraph context if missing

    7. Context Building

    8. System prompt combined with HtmlGraph context
    9. Token counting to ensure within budget

    10. Truncation at newline boundaries if needed

    11. Hook Output

    12. Valid JSON with structure: 

      {
  "continue": true,
  "hookSpecificOutput": {
    "hookEventName": "SessionStart",
    "additionalContext": "system prompt + htmlgraph context",
    "source": "startup|resume|compact|clear",
    "session_id": "session-id"
  }
}

    13. Claude Receives Context

    14. additionalContext injected into Claude's system context
    15. Persists throughout session
    16. Survives compact/resume cycles
